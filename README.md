# 📊 Task 04 - Web Data Scraper

A modern Python application developed using **Tkinter** that retrieves book information from the Open Library API and exports the collected data into a CSV file.

## 🚀 Features

- 🔍 Search books using any keyword
- 📖 Fetch book title, author, publication year, rating, and genre
- 🖥️ Modern Tkinter GUI
- 📋 Display results in a structured table
- 📁 Export scraped data to CSV
- ⚡ Fast and responsive interface using multithreading
- 🎨 Clean dark-themed UI

## 🛠 Technologies Used

- Python
- Tkinter
- Open Library API
- urllib
- CSV
- Threading
